About SalesCheat Bot

SalesCheat Bot is a Microsoft Teams integration designed to give recruiters real-time support on the sales floor. It generates instant intel on candidates — called Battle Cards — to help craft personalized, high-converting outreach.

Technical Stack

Node.js Express Bot Framework SDK Vector Database (JSON) Cosine Similarity TF-IDF Embeddings Adaptive Cards NLP Parsing

Architecture

  • Profile Parser — Extracts structured data from URLs and free text
  • Battle Card Engine — Rule-based generator using an industry knowledge base
  • Vector Store — JSON-based vector DB with TF-IDF embeddings
  • Template Matcher — Cosine similarity matching with metadata bonuses
  • Teams Bot Adapter — Microsoft Bot Framework for Teams deployment

Data

  • 20+ curated outreach templates with performance scores
  • 8 industries x 4 seniority levels = 32 pain point clusters
  • 100+ dimension keyword vocabulary for vector embeddings
